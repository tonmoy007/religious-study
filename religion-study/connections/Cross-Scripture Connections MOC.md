# 🔗 Cross-Scripture Connections

Evidence-based comparisons that connect **stories, figures, and teachings**
*across* the scriptures now in the **[[Scriptures Library]]**. Unlike the
conceptual overview in [[Connections Between Religious Texts]], every note here
**quotes the actual texts** side by side and links to the source passages so
you can read them in context.

> [!tip] How these notes work
> Each note takes one figure, story, or motif and lays the parallel passages
> next to each other — same event, different scripture — then asks what is
> *shared*, what is *changed*, and *why*. The quotes are verbatim from the
> public-domain translations in the library.

## How to Read a Connection Honestly
Borrowed from [[Connections Between Religious Texts]] — keep three explanations distinct:
1. **Genealogical** — one text demonstrably borrowed from another (the Qur'an *knowingly* retells biblical figures).
2. **Diffusion** — ideas spread through cultural contact (Golden Rule along trade routes).
3. **Independent / archetypal** — shared human experience yields similar forms (flood myths worldwide).

> Beware *parallelomania*: surface similarity is not shared meaning. For each
> parallel below, the note flags which kind of connection it most likely is.
